df = pd.read_csv('../../train.csv')

# 2. Crear la variable de respuesta categrica basada en los terciles del precio de venta
tercil_1 = df['SalePrice'].quantile(1/3)
tercil_2 = df['SalePrice'].quantile(2/3)

# Crear la categora de precio
df['PriceCategory'] = pd.cut(
    df['SalePrice'], 
    bins=[0, tercil_1, tercil_2, float('inf')], 
    labels=['Econmica', 'Intermedia', 'Cara']
)

# 3. Seleccionar las caractersticas para el modelo
selected_features = [
    'OverallQual',    # Calidad general
    'GrLivArea',      # rea habitable
    'GarageCars',     # Capacidad del garaje
    'YearBuilt',      # Ao de construccin
    'TotalBsmtSF',    # Superficie total del stano
    'FullBath',       # Baos completos
    'Neighborhood',   # Vecindario (categrica)
    'ExterQual',      # Calidad del exterior (categrica)
    'KitchenQual',    # Calidad de la cocina (categrica)
    'BsmtQual'        # Calidad del stano (categrica)
]

# 4. Preparar los datos
X = df[selected_features].copy()
y = df['PriceCategory']

In[68]:
# Manejar valores faltantes en caractersticas numricas
numeric_features = X.select_dtypes(include=['int64', 'float64']).columns
imputer = SimpleImputer(strategy='median')
X[numeric_features] = imputer.fit_transform(X[numeric_features])

# Manejar valores faltantes en caractersticas categricas
categorical_features = X.select_dtypes(include=['object']).columns
X[categorical_features] = X[categorical_features].fillna('None')

# Objetos de preprocesamiento para pipeline de modelos
preprocesador_categorico = OneHotEncoder(handle_unknown="ignore")
preprocesador_numerico = StandardScaler()

# Dividir los datos en conjunto de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

Aunque ambos modelos muestran un rendimiento similar, el Modelo 2 es la mejor opcin en este caso. No solo obtiene mejores mtricas globales y por clase, sino que tambin lo hace de forma ms eficiente. Su arquitectura ms simple y moderna (una sola capa con relu) parece adaptarse mejor a la estructura del conjunto de datos sin comprometer la generalizacin.
